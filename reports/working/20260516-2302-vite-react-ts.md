## 3. 検討・判断したこと

- `app/web` を Vite のアプリルートとし、既存 CSS と画像アセットは維持した。
- Vite production build で静的画像が解決されるよう、画像を `app/web/public/assets/` 配下へ移動し、React 側は `/assets/...` で参照する方針にした。
- 既存の大きな `app.js` は、画面コンポーネント、コースデータ、canvas 描画、レース制御 hook に分けた。
- オンライン参加、共有、ランキングなど未実装機能は、従来どおり未接続・準備中として表示する方針を維持した。

## 4. 実施した作業

- `package.json`、`package-lock.json`、`vite.config.ts`、`tsconfig*.json` を追加した。
- `index.html` を React root と Vite entry に更新した。
- メニュー、チーム作成、コード参加、マップ一覧、待機、ゲーム、リザルト画面を `src/components/screens/` に分離した。
- ボタン、画面 wrapper、モーダルなどを `src/components/ui/` に分離した。
- コースカード、コース詳細、コースパーツを `src/components/course/` に分離した。
- コースマスタを `src/data/courses.ts`、画像パスを `src/data/assets.ts` に分離した。
- canvas のトラック・描画・画像 preload を `src/game/`、カウントダウンとレースループを `useRaceController` に分離した。
- README と UI spec docs の旧 `app.js` / `app/web/assets` 参照を更新した。

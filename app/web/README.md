# ビリビリレーサーズ Web 版

`app/web` は、ビリビリ棒のように壁へ触れるとスタートへ戻る 2D レーシングゲームの一人プレイ初版です。

## 起動方法

リポジトリルートから次のコマンドで静的ファイルを配信します。

```bash
python3 -m http.server 4173 --directory app/web
```

ブラウザで `http://127.0.0.1:4173/` を開きます。

## 操作

- `WASD` または `矢印キー`: カート移動
- `Space`: ブースト
- `Enter`: レース開始前画面で即スタート
- `Esc`: モーダルを閉じる、またはルーム作成画面からメニューへ戻る

## 画面

- メニュー画面
- ルーム作成画面
- ゲーム開始時画面
- ゲーム中画面
- リザルト画面

各画面の構造、表示文言、操作、状態、視覚表現は [`UI_DESCRIPTION.md`](./UI_DESCRIPTION.md) にまとめています。

# ビリビリレーサーズ一人プレイ版

- 状態: doing
- タスク種別: 機能追加
- 作成日時: 2026-05-16 16:05

## 背景

複数人で遊ぶ 2D マップのレーシングゲーム構想に対して、まずは `app/web` 配下に一人で遊べる Web 版を作る。壁に触れるとアウトになりスタートへ戻され、先にゴールするルールを初版では一人用の完走体験として実装する。

## 目的

`.workspace` の参照 UI/UX を踏まえ、メニュー画面、ルーム作成画面、ゲーム開始時の画面、ゲーム中の画面、リザルト画面を持つ、起動可能な一人プレイ版を提供する。

## スコープ

- `app/web` 配下の静的 Web アプリ実装
- Canvas による 2D コース、車両、壁接触、スタート復帰、ラップ、ゴール処理
- `.workspace` 参照に基づく明るいアーケード風 UI/UX
- 作業レポートの作成

## スコープ外

- 実ネットワーク対戦
- 永続的なルーム管理
- サーバー API
- 本番ランキングやフレンド機能

## 実装方針

- 依存インストールなしで確認できるよう、HTML/CSS/JavaScript の静的アプリとして構成する。
- UI は React などのランタイムに依存せず、状態遷移と Canvas 描画を小さく実装する。
- 複数人要素は将来拡張を想起できる表示に留め、未実装機能は「準備中」と明記する。
- `.workspace` の白い太枠パネル、青/ピンク/緑/黄ボタン、電撃コース、周辺 HUD、リザルト演出を CSS と Canvas で再現する。

## ドキュメント保守計画

新規アプリの起動方法と操作方法は `app/web/README.md` に記載する。既存 README は存在しないため、リポジトリ全体への追記は行わない。
